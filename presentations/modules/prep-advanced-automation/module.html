Pre-Workshop Instructions

Please perform the following steps before the workshop. Some of these steps take a few minutes and you may have issues specific to your machine that we can address in advance.
Email me if you get stuck.

STEP 1: Install the latest Cytoscape

  • Download Cytoscape v3.7.0
  • Open and follow installation steps

Pre-Workshop Instructions

STEP 2: Install R, RStudio and Bioconductor

  • You will need R version 3.5.0 or greater
  • You will need RStudio 1.1 or greater
  • You will need Bioconductor 3.7. Simply run these R commands: 
    install.packages("BiocManager")
BiocManager::install()

Pre-Workshop Instructions

STEP 3: Install the RCy3 package

  • OPTION 1: Given that you have the latest Bioconductor installed, you can simply run this R command: 
    BiocManager::install("RCy3")
  • OPTION 2: If you run into troubles or want an alternative to Bioconductor, then you can attempt installing the development code
  • Additional troubleshooting tips are also available
  • And Python lovers might want to install py2cytoscape

Pre-Workshop Instructions

STEP 4: Take it for a test drive

  • Open the RCy3 vignettes with this R command: 
    browseVignettes("RCy3")
  • Click on the HTML link for 1. Overview of RCy3
  • Go through Steps 1 through 4 of the overview vignette before the workshop

    • Pro-tip: You can copy/paste code chunks from the HTML web page into RStudio or save/open the Rmd script provide by the source link

That's it! If the initial steps of the overview vignette ran without warnings or errors and you ended up a 4-node network in Cytoscape, then you are ready for the workshop.
