UI improvements

Switch to Cytoscape 3.8 and demo each of these

Performance improvements

  • Phase 1 of the network renderer rework
    • Significantly faster, especially for larger networks
      • Now interactive for networks ~10,000 nodes, ~3,000,000 edges
    • Annotations behave much, much better
    • Lays the foundation for future enhancements
  • Performance optimization of some table data
# Point out that BigNetwork has 9,000 nodes and almost 4 million edges. It is interactive on a workstation with 32Gb of RAM.

Java upgrade

  • Support for Java 11
  • Specifically tested against OpenJDK 11 (no Oracle license restrictions)
  • Also included JavaFX 13
# Mention that this was a *lot* of work

Core App Updates

Analyze Network
The old NetworkAnalyzer has been completely rewritten and simplified. The results may still be found in the node and edge tables as before
Network Merge
The network merge code has been significantly updated
CyBrowser
The update to JavaFX 13 has resulted in a significant functionality boost for CyBrowser, fixing many bugs. It's still not a replacement for your daily browsing, but it's better than before
CyNDEx
CyNDEx no longer depends on JXBrowser, which should significantly reduce the startup and implementation issues.
