Importing Network Data

    Cytoscape can import network data from:

    • Public repositories:
      • EBI/PSICQUIC
      • STRING (via the stringApp)
      • Reactome (via the ReactomeFI App)
      • Wikipathways (via the Wikipathways App)
      • Pathway Commons (via the CyPath2 App)
