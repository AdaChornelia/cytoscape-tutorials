Loading Data

  • Cytoscape can import network data from:
    • Files (or URLs)
      • Excel, TSV, CSV
      • eXtensible Graph Markup and Modelling Language: XGMML
      • Systems Biology Markup Language (SBML)
      • BioPAX
      • PCI-MI
      • Simple Interaction Format: SIF
      • Graph Markup Language: GML
      • ... and others depending on loaded Apps

* Bold: loads table data also

Loading Data

  • Cytoscape can import network data from:
    • Public repositories:
      • EBI/PSICQUIC
      • STRING (via the stringApp)
      • Reactome (via the ReactomeFI App)
      • Wikipathways (via the Wikipathways App)
      • Pathway Commons (via the CyPath2 App)
