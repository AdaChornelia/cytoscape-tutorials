Tour of Cytoscape

This tutorial represents a high-level tour of the features of Cytoscape. Detailed instructions for specific features are provided in separate tutorials linked to from each section.

This tutorial is designed to run as a demo in the CyBrowser app, with interactive buttons for each task.

Data

Cytoscape allows the user to add arbitrary node, edge and network information to Cytoscape as node/edge/network data columns. This could include annotation data for genes, protein-protein interaction confidence scores or transcriptomics data.

Data is visible in the Table Panel and can be imported under File → Import Table. Data files can be local or can be loaded from a URL or online database.

For details, see Loading Omics Data tutorial.

For details, see Loading Omics Data tutorial.

Styles

One of Cytoscape’s strengths in network visualization is the ability to allow users to encode any table data (name, type, degree, weight, expression data, etc.) as a property (such as color, size of node, transparency, or font type) of the network. A set of these encoded or mapped table data sets is called a Style and is controlled in the Style tab of the Control Panel.

Layout

Networks in Cytoscape can be manually manipulated in terms of layout, but Cytoscape also supports a large collection of automatic layout algorithms. Layout algorithms are available under the Layout menu.

Network manipulation

Networks can be filtered based on associated data or network properties, and can be edited and manipulated. The Network Search Bar is used to search for individual nodes and edges. Filtering of nodes is done in the Select tab of the Control Panel.

For details, see Filtering and Editing tutorial.


For details, see Filtering and Editing tutorial.

Apps

Cytoscape’s capabilities are not fixed but can be expanded in a variety of ways with apps. The Cytoscape App Store organizes over 300 apps in categories such as data integration, online data/network import, network analysis and visualization.

The App Manager is available in Cytoscape under Apps → App Manager....

For details, see App Manager manual.

Export

Cytoscape provides a number of ways to save results and visualizations:

  • As a session: File→Save, File→Save As...
  • As an image: File→Export as Image...
  • To the web: File→Export as Web Page...
  • To a public repository: File→Export→Network Collection to NDEx
  • As a graph format file: File→Export→Network.
    Formats:
    • CX JSON
    • Cytoscape.js JSON
    • GraphML
    • PSI-MI
    • XGMML
    • SIF
