Tour of Cytoscape

This tutorial represents a high-level tour of the features of Cytoscape. Detailed instructions for specific features are provided in separate tutorials linked to from each section.

Cytoscape accepts network data in a variety of formats and from multiple sources.

Data

Cytoscape allows the user to add arbitrary node, edge and network information to Cytoscape as node/edge/network data columns. This could include, annotation data for genes, protein-protein interaction confidence scores or transcriptomics data.

Styles

One of Cytoscape’s strengths in network visualization is the ability to allow users to encode any table data (name, type, degree, weight, expression data, etc.) as a property (such as color, size of node, transparency, or font type) of the network. A set of these encoded or mapped table data sets is called a Style.

For details, see Mapping Data tutorial.

Network manipulation

Networks can be filtered based on associated data or network properties, and can be edited and manipulated.

Analysis

With the NetworkAnalyzer, it is possible to compute a comprehensive set of topological parameters for undirected and directed networks.

Apps

Cytoscape’s capabilities are not fixed but can be expanded in a variety of ways with apps. The Cytoscape App Store organizes over 300 apps in categories such as data integration, online data/network import, network analysis and visucalization.
